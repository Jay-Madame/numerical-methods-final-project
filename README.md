# To Start

1. Run the following command in terminal:
```python -m venv .venv```

2. With care to which OS you are using, run the following command to activate the virtual environment

**macOS/Linux**:
```source .venv/bin/activate```

**windows** command prompt:
```.venv\Scripts\activate.bat```

**windows** powerShell:
```.\.venv\Scripts\Activate.ps1```

<br>
Now, you should be able to run main.py from terminal or IDE without compilation errors.
